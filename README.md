# 3 Body Problem - Fan Project

This is a fan-made web project dedicated to the "3 Body Problem" series, developed as part of a university project. The site provides an immersive experience into the world of Trisolaris, the Oxford Five, and the terrifying reality of the countdown.

## Project Overview

The website features a dark, atmospheric design with glitch effects and animations to match the themes of the series. It includes several sections detailing different aspects of the story and its scientific concepts.

## Key Features

- **Immersive Interface**: High-quality visual effects, including a loading screen with a countdown timer, scanlines, and glow effects.
- **Science Section**: Information about the Oxford Five, the Sophon effect, and nanofiber technology.
- **The Enemy**: Detailed background on the San-Ti and their civilization.
- **Characters**: A grid layout showcasing the main characters from the series with interactive hover effects.
- **The Game**: A dedicated page featuring a 3D STL model simulation with interactive rotation and environment lighting.
- **Responsive Design**: Fully optimized for both desktop and mobile viewing.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS with custom animations)
- JavaScript (ES6+)
- Three.js (for 3D model rendering)

## Site Structure

- `index.html`: The home page featuring "The Countdown" and a link to initiate research.
- `science.html`: Exploration of breaking scientific laws and key technological breakthroughs.
- `enemy.html`: Information about the Trisolaran threat.
- `characters.html`: Profiles of the key figures in the struggle for humanity.
- `game.html`: Interactive 3D simulation of the Three Body system components.
